# Static Gallery Images

Upload image files into this folder, then list them in `gallery.json`.

Example:

```json
[
  {
    "src": "assets/gallery/image-01.jpg",
    "title": "Image 1",
    "alt": "Short description of image 1",
    "caption": "Optional caption shown under the title"
  }
]
```

Use simple filenames without spaces when possible, such as `image-01.jpg`.
