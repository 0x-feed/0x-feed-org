// recommended starting order · for first-time learners
  1. Start fundamentals: Professor Messer (Network+ → Security+) is the single best free study channel. Pair with SANS Cyber Aces.
  2. Build hands-on: TryHackMe's free pre-security path, then OverTheWire Bandit for the Linux command line.
  3. Practice CTFs: picoCTF Gym is the largest free archive. Solve old problems, watch writeups.
  4. Specialize: web → PortSwigger Academy. blue team → RangeForce Community. cloud → Microsoft Learn or AWS Skill Builder.
  5. Earn a credential: Security+ via Professor Messer + Cybrary. Optional: Google Cybersecurity Cert (Coursera, financial aid available).
